-> ls (ver archivos dentro de directorio) ( [-al] para mostrar los archivos en formade lista)
-> cd [nombre carpeta] (para ir a este directortio) ( [..] para regresarse una carpeta antes)
-> pwd (para mostrar directorio en el que estoy actualmente)
-> mkdir (Make directory basicamente crear una carpeta)
-> touch [nombre] (crear archivo colocando nombre y tipo de archivo como [.txt] )
-> cat [nombre] (mostrar contenido del archivo colocando nombre y tipo de archivo como [.txt] )
-> history (para ver historial de comandos)(luego se puede colocar [!] y el numero para ejecutar ese comando)
-> clear (para limpiar pantalla) (tambien con [ctrl + l])
-> rm [nombre] (borrar archivo)
-> --help (muestra todos los comandos con explicaciones)
